# ChatGPT Bot

This bot uses data from ChatGPT to generate responses to user input, create files and folders, and perform tasks such as refactoring and bug finding.

## Installation

To install the dependencies for this project, run the following command in your terminal:
pip install -r requirements.txt

## Usage

To run the bot, use the following command in your terminal:

python main.py


This will start the bot, which will prompt you for input and generate a response using data from ChatGPT.
